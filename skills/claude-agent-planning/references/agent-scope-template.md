# Agent Agent Scope Template

This reference supports the delegated agent workflow lifecycle.

Use this template to plan a delegated agent workflow before design starts.

1. Name the orchestration pattern and its overall goal.
2. Define each role that will participate in the workflow.
3. Assign clear ownership boundaries to each role.
4. Record expected handoffs, merge points, and escalation paths.
5. List dependencies such as tools, repos, or external services.
6. Capture one happy-path orchestration and one failure case.
7. Define the final deliverable expected from the orchestrator.
8. Note constraints on parallelism, isolation, and coordination.
9. Document portability concerns across agent runtimes.
10. Do not proceed while any role boundary remains ambiguous.
