## GPT Models

| Model | Best for |
| --- | --- |
| `gpt-4o` | Flagship multimodal — text, vision, audio |
| `gpt-4o-mini` | Fast, affordable tasks |
| `gpt-4-turbo` | High-intelligence text tasks |
| `gpt-3.5-turbo` | Legacy, fast and cheap |

## Reasoning Models (o-series)

| Model | Notes |
| --- | --- |
| `o1` | Advanced chain-of-thought reasoning |
| `o1-mini` | Faster, cheaper reasoning |
| `o3` | Latest, most capable reasoning |
| `o3-mini` | Fast reasoning |
| `o4-mini` | Fast, efficient reasoning |

## Model Selection Guide

- **General assistant / chat**: `gpt-4o`
- **Fast / low-cost**: `gpt-4o-mini`
- **Complex multi-step reasoning**: `o1` or `o3`
- **Embeddings**: `text-embedding-3-large`
