# Agent Phase Planning

This reference supports the delegated agent workflows SDLC.

Planning creates the shared source of truth for the workflow family.

1. Document purpose, audience, role boundaries, and explicit non-goals.
2. Record handoffs, outputs, and execution constraints.
3. Capture portability expectations for other runtimes.
4. Record at least one success scenario and one recovery scenario.
5. Keep scope narrow enough to validate in one pass.
6. Do not move forward with TBD entries in the planning packet.
7. Use the corresponding planning skill when refining these artifacts.
8. Treat planning changes as contract changes for downstream phases.
9. Record assumptions that need later verification.
10. Planning is the anchor for every later gate.
